Deactivate hotbar=Désactiver la barre d'outils
WARNING: If you enable this setting, AdvancedCompany won't touch the hotbar and inventories at all. This means that the inventory perk gets auto deactivated, all equippable items, including cursed items, will get forcefully deactivated, the energy bars are removed, customization of the hotbar is removed, there wont be any hotkeys, neither flashlight nor quick access, added and no inventory fixes like flashlight fixes. This mode is experimental and there won't be any support for it whatsoever. If I see you using it and creating a GitHub issue relating to this mode it will get auto closed. Activating this option might result in unwanted results as the hotbar changes are a core function of AdvancedCompany. Spawning in and equipping any equipment item might break your game in unexpected ways.=ATTENTION : Si vous activez ce paramètre, AdvancedCompany ne touchera pas du tout à la barre de chauffe et aux inventaires. Cela signifie que la perk d'inventaire est automatiquement désactivée, que tous les objets équipables, y compris les objets maudits, seront désactivés de force, que les barres d'énergie sont supprimées, que la personnalisation de la barre de chauffe est supprimée, qu'il n'y aura pas de raccourcis clavier, ni de lampe de poche, ni d'accès rapide, ajoutés et qu'il n'y aura pas de correction d'inventaire comme celle de la lampe de poche. Ce mode est expérimental et il n'y aura pas de support pour lui. Si je vois que vous l'utilisez et que vous créez un problème GitHub relatif à ce mode, il sera automatiquement fermé. L'activation de cette option peut entraîner des résultats indésirables, car les modifications de la barre d'outils sont une fonction essentielle d'AdvancedCompany. Le fait d'entrer dans le jeu et d'équiper n'importe quel élément d'équipement peut perturber votre jeu de manière inattendue.

#Avantages
Perks=Avantages

Here you can configure which perks to activate. All defined base values will get applied regardless of the active state of the perk, enabling you to use the perks just as a way to tweak your game without using perks.\n\nThe costs are a comma-separated list containing the XP costs for each level. The max level of a perk is determined by the count of values provided.=Ici, vous pouvez configurer quels avantages activer. Toutes les valeurs de base définies seront appliquées indépendamment de l'état actif de l'avantage, vous permettant d'utiliser les avantages comme une façon d'ajuster votre jeu sans les utiliser.\n\nLes coûts sont une liste séparée par des virgules contenant les coûts en XP pour chaque niveau. Le niveau maximum d'un avantage est déterminé par le nombre de valeurs fournies.
